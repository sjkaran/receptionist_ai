## What is this?
An AI enabled trainable voice engine to be used as in Reception of any office, which is trained by the user to greet and manage the customers at the office reception.

## What am I using here?
> It is totally made using python and its frameworks like:
    * sounddevice - for recording/listening to voice
    * numpy - for processing the recorded data
    * speech_recognition - to recognize the recorded voice
    * sqlite - to store the training data
    * json - as temporary relative data storage
    * google.gnerativeai - for AI api integration (currently not working)
    * threading - for multithreading (to make the recording and user interaction faster)
